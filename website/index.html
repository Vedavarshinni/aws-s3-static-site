🚀 AWS DevOps CI/CD Project

Static website deployed on AWS S3 using Terraform Infrastructure as Code and automated through GitHub Actions CI/CD Pipeline.

🔧 Technologies Used

  • AWS S3
  • Terraform
  • GitHub Actions
  • Infrastructure as Code (IaC)
  • CI/CD Pipeline
  • HTML & CSS

📌 Project Workflow

GitHub → GitHub Actions → Terraform → AWS S3 Deployment

🌱 Learning Goals

Building hands-on DevOps skills by automating cloud infrastructure deployment using modern CI/CD practices.
